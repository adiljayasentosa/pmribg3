Setup Firebase · PMR WIRA UNIT SMK IBG 3

Jalankan halaman ini sekali saja untuk menginisialisasi database Firestore dan membuat akun pengguna.

Memeriksa konfigurasi Firebase…
1

Buat Project Firebase

Buka console.firebase.google.com → Add project → beri nama (misal: pmr-smkibg3) → selesaikan wizard.

2

Aktifkan Firestore & Authentication

Di sidebar Firebase Console: Firestore Database → Create database (pilih mode production atau test) · Authentication → Sign-in method → aktifkan Email/Password.

3

Salin Config ke firebase-config.js

Project Settings → Your apps → Add app (Web) → salin objek config → tempel ke js/firebase-config.js, lalu ubah FIREBASE_ENABLED = true.

4

Atur Firestore Rules

Firestore → Rules → salin rules berikut:

rules_version = '2';
service cloud.firestore {
  match /databases/{database}/documents {
    // Hanya user yang sudah login yang bisa baca/tulis
    match /{document=**} {
      allow read, write: if request.auth != null;
    }
  }
}
5

Jalankan Seed & Buat Akun (tombol di bawah)

Klik tombol di bawah secara berurutan. Pastikan sudah login sebagai admin Firebase terlebih dahulu atau gunakan mode test rules sementara.

Menunggu tindakan…
⚠️ Perhatian: Halaman setup hanya perlu dijalankan sekali. Menjalankan seed berulang kali bisa menyebabkan data duplikat di Firestore. Hapus file setup.html setelah selesai untuk keamanan.
